# 本地卡背预设 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 单面卡在右侧本地渲染视图设置发布用卡背预设，英文和中文参考图保持只读。

**Architecture:** 后端在映射索引中为单面卡 face 记录 `back_override`，并提供管理员 API 保存、清除和读取预设。前端只在单面卡的本地渲染背面卡片显示选择器；发布替换读取该索引作为中文 TTS 包写入依据。

**Tech Stack:** FastAPI、SQLAlchemy async、React、TypeScript、Ant Design、JSON 映射索引。
